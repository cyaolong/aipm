# 01 Idea Enhanced

## 一句话定位
面向产品经理的 AI 方案生成器：输入 idea 即可自动完善构想、输出标准 PRD，并生成可复制可编辑的 iOS 26 低保真原型。

## 目标用户与场景
- 目标用户：中小团队/独立产品经理、创业 PM、咨询/外包 PM。
- 核心场景：
  - 临时接到新项目，需要快速形成可沟通的方案与 PRD。
  - 需要快速验证想法质量，形成对外展示材料。
  - 在缺少设计资源情况下生成可执行的原型参考。

## 市场定位
- 定位为“PM 工作流自动化工具”，强调文档与原型一体化产出。
- 与通用 AI 文档工具区别在于：结构化 PRD + 交互/页面清单 + Figma 原型联动。

## 竞品与差异化
- 竞品（泛化）：通用 LLM、文档模板工具、原型工具。
- 差异化：
  - 一键生成“完善构想 + PRD + 原型”，形成闭环。
  - MVP 规则约束，减少冗余内容，提高可执行性。
  - 输出结构固定且可复用，降低 PM 成本。

## 价值主张
- 降低 PM 从 idea 到交付物的时间成本。
- 提供结构化与可复制的 PRD 和原型，提高沟通效率。
- 通过订阅 + 免费额度建立持续使用习惯。

## 核心场景与路径
1) 用户在网页平台注册/登录 → 进入工作台
2) 输入 idea → 系统自动补全并给出方案摘要
3) 生成标准 PRD → 用户快速调整
4) 生成 Figma 低保真原型 → 直接分享/协作

## MVP 范围
- 做：
  - 网页平台注册/登录与全流程操作
  - 账号体系与历史项目管理
  - 订阅计费 + token 额度消耗
  - 结构化 PRD（标准深度）
  - Figma 低保真原型链接输出（可复制、可编辑）
- 不做：
  - 移动端 App/小程序
  - 高保真视觉设计
  - 复杂的多人协作流程

## 关键假设与待验证
- 付费意愿：月付 20 元/年付 120 元能被接受。
- 产出质量可达到“可用 PRD + 可协作原型”的最低门槛。
- token 额度策略能促成转化，而不导致流失。
