# 03 原型说明

## 本地高保真原型（HTML）
- 原型路径：`Ideas/pm_agent/prototype/index.html`
- 说明：原型覆盖完整流程与状态，不依赖 Figma 绘制，便于全自动演示与评审。

## 页面清单（线框对应）
- 登录/注册：完成账号创建与登录 / 模块：登录表单、注册表单、协议提示 / 入口与出口：入口=落地页CTA，出口=工作台
- 工作台首页：展示项目概览与入口 / 模块：项目列表、空状态、主CTA / 入口与出口：入口=登录后，出口=项目详情/新建项目
- 新建项目（Idea 输入）：收集 idea 与基础信息 / 模块：输入区、提示文案、生成按钮 / 入口与出口：入口=工作台CTA，出口=生成结果
- 构想与 PRD 结果页：展示完善构想与 PRD / 模块：分段内容、复制、编辑区 / 入口与出口：入口=生成完成，出口=生成原型
- 原型生成页：展示生成进度与链接 / 模块：进度、链接、复制按钮 / 入口与出口：入口=结果页CTA，出口=项目详情
- 项目详情：历史版本与产出汇总 / 模块：版本列表、摘要、链接区 / 入口与出口：入口=工作台列表，出口=结果页/原型页
- 订阅与额度：套餐选择与 token 消耗 / 模块：套餐卡片、支付CTA、额度进度 / 入口与出口：入口=设置/额度提示，出口=工作台

## 核心流程
1) 登录/注册 → 进入工作台
2) 新建项目 → 输入 idea → 生成构想与 PRD
3) 生成原型 → 获取 Figma 链接 → 回到项目详情
4) token 用尽 → 引导订阅 → 返回工作台继续使用

## 关键组件（线框）
- 列表、卡片、按钮、输入框、分段切换、进度条、空状态、错误提示

## 交互粒度规范
- 页面级：主入口 CTA（新建项目/生成/订阅）、导航返回路径
- 模块级：表单校验、列表加载、空/错误/无数据
- 状态级：默认/加载中/成功/失败

## 交互说明
- 新建项目：输入 idea 后点击“生成”，进入加载状态，完成后跳转结果页
- 结果页：支持在构想与 PRD 间切换；点击“生成原型”进入原型生成
- 原型生成：生成完成展示可复制/可编辑的 Figma 链接
- 订阅：额度不足时弹出升级提示，订阅成功返回工作台

## 状态覆盖
- 每页覆盖空/错误/加载状态
